# 🚀 CareerCraft

CareerCraft is a modern SaaS-style career assistant web application designed to help job seekers manage their job search, build ATS-friendly CVs, and leverage AI tools for career growth.

---

## ✨ Features

### 📊 Dashboard
- Overview of job search activity
- Application tracking
- Saved jobs preview
- Interview preparation tips
- Profile strength indicator

---

### 📄 CV Builder
- Create and edit professional CVs
- Multiple templates (Modern, Minimal, European)
- Drag & drop CV sections
- ATS score analysis
- Export CV to PDF

---

### 🔍 Job Search
- Browse job opportunities
- Filter and explore roles
- Save jobs for later review

---

### 📌 Applications Tracker
- Track job applications
- Manage statuses (Saved, Applied, Interview, Offer, Rejected)
- Visual pipeline board

---

### 🤖 AI Assistant
- Ask career-related questions
- Generate cover letters
- Get CV improvement suggestions

---

### 🎯 Interview Preparation
- Practice behavioral & technical questions
- STAR method builder
- Interview checklist

---

### ⚙️ Settings
- Theme switching (Light / Dark)
- Language support (EN, DE, AR, NL)

---

## 🛠️ Tech Stack

- ⚛️ React (Vite)
- 🎨 Tailwind CSS
- 🎞️ Framer Motion
- 🧠 Custom Hooks & Context API
- 📄 jsPDF + html2canvas (PDF Export)
- 📊 Data Visualization (Charts)
